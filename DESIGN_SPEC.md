# 设计规范

## 一、核心风格定位

-   **整体风格流派**：B端企业级SaaS风格 (Enterprise SaaS)、现代科技商务风 (Modern Corporate Tech)、极简主义 (Minimalism)。
-   **设计哲学和情感基调**：专业、严谨、高效、可信赖。通过大面积留白和冷色调（蓝色系）传达科技感与理性，避免过度装饰，强调信息层级和阅读效率。
-   **行业属性和适用场景**：适用于软件开发、数据分析、云计算、企业服务等B端科技公司的产品官网、SaaS平台落地页或后台管理系统首页。

## 二、精确颜色系统

-   **主色**：`#1E4696`（博彦科技品牌深蓝，用于Logo、核心强调）及 `#2B65EC`（科技亮蓝，用于按钮、链接、高亮图标，使用比例 15%）。
-   **辅助色1**：`#E8F0FE`（极浅蓝，用于背景块、Hover状态、标签背景，使用比例 10%）。
-   **辅助色2**：`#F0F2F5`（浅灰蓝，用于页面大背景、分割区域，使用比例 20%）。
-   **中性色**：
    -   背景：`#FFFFFF`（卡片/内容区），`#F5F7FA`（页面底色）。
    -   文字：`#1D2129`（主标题/重要信息），`#4E5969`（正文/次要信息），`#86909C`（辅助说明/占位符）。
    -   分割线：`#E5E6EB`（极淡灰，用于卡片边框或区域分割）。
-   **渐变规则**：
    -   主按钮/底部Banner：线性渐变，从左上到右下 (`135deg`)，`#2B65EC` 到 `#1E4696`。
    -   装饰元素：径向渐变，中心 `#5B8FF9` (透明度 0.2) 到边缘透明。

## 三、排版系统

-   **字体家族**：无衬线字体 (Sans-serif)。首选 `PingFang SC`, `Microsoft YaHei`, `Helvetica Neue`, `Arial`。
-   **字号层级**：
    -   大标题 (Hero Title)：36px - 48px。
    -   模块标题 (Section Title)：24px - 28px。
    -   卡片标题：16px - 18px。
    -   正文 (Body)：14px。
    -   小字/注释：12px。
    -   按钮文字：14px - 16px。
-   **字重分布**：
    -   粗体 (Bold/700)：仅用于大标题和核心数据强调。
    -   中等 (Medium/500)：用于模块标题、卡片标题、按钮文字。
    -   常规 (Regular/400)：用于正文、辅助说明。
-   **行高和字间距**：
    -   标题行高：1.2 - 1.4。
    -   正文行高：1.5 - 1.6 (保证阅读舒适度)。
    -   字间距：默认 0，英文大写字母间距 +1px。
-   **对齐方式**：全局左对齐为主，Hero区域标题可左对齐或居中，导航栏右对齐，底部Footer居中。

## 四、组件样式规范

-   **按钮**：
    -   形状：矩形，小圆角。
    -   圆角半径：4px - 6px (B端常用，显得严谨)。
    -   边框：无 (实心按钮) 或 1px solid `#2B65EC` (幽灵按钮)。
    -   阴影：`0 2px 4px rgba(43, 101, 236, 0.2)` (轻微投影)。
    -   悬停效果：背景色加深 10%，或阴影加深。
-   **卡片**：
    -   圆角：8px - 12px。
    -   阴影：`0 4px 12px rgba(0, 0, 0, 0.04)` (非常柔和的漫射阴影，营造悬浮感)。
    -   内边距：24px - 32px。
    -   边框：1px solid `#F0F2F5` (极淡边框，增加边界感)。
    -   背景透明度：纯白 `#FFFFFF` 或 95% 透明度 (若背景有纹理)。
-   **输入框** (若有)：
    -   边框样式：1px solid `#E5E6EB`。
    -   圆角：4px。
    -   聚焦效果：边框变蓝 `#2B65EC`，带 `0 0 0 2px rgba(43, 101, 236, 0.1)` 的光晕。
-   **导航栏**：
    -   高度：64px - 80px。
    -   背景：纯白 `#FFFFFF` 或 毛玻璃效果 (Backdrop-filter blur)。
    -   分割线：底部 1px solid `#F0F2F5`。
-   **图标**：
    -   风格：2.5D等距视角 (Isometric) 或 双色扁平风格 (Duotone)。
    -   粗细：线条图标 1.5px - 2px。
    -   大小：24px (行内), 48px-64px (展示用)。
    -   配色：主色蓝 + 浅蓝/灰色。

## 五、布局与空间

-   **网格系统**：12列栅格系统 (Grid System)。
-   **Gutter宽度**：24px。
-   **页面边距**：左右边距 (Margin) 至少 40px - 80px，最大内容宽度 (Max-width) 约 1200px - 1440px，居中显示。
-   **元素间距比例**：
    -   模块间距 (Section Gap)：80px - 120px (大留白)。
    -   卡片间距：24px。
    -   标题与正文间距：16px - 24px。
-   **留白程度**：高留白 (High Whitespace)，呼吸感强，避免拥挤。

## 六、特殊视觉效果

-   **阴影风格**：全局使用软阴影 (Soft Shadow)，无硬阴影。阴影颜色偏蓝灰色 `rgba(30, 70, 150, 0.08)` 而非纯黑，使界面更通透。
-   **毛玻璃效果**：顶部导航栏或悬浮卡片使用 `backdrop-filter: blur(12px)`，背景色 `rgba(255, 255, 255, 0.8)`。
-   **动画风格**：
    -   过渡时长：0.3s。
    -   缓动函数：`ease-in-out` 或 `cubic-bezier(0.4, 0, 0.2, 1)`。
    -   动画类型：Hover时轻微上浮 (`translateY(-4px)`)，淡入淡出。
-   **纹理和装饰元素**：
    -   背景纹理：极淡的网格点阵 (Dot Pattern) 或 几何线条，透明度 5%。
    -   装饰图形：顶部Hero区域使用抽象的3D球体或流体形状（蓝紫色调），增加科技感。
    -   客户Logo墙：统一处理为灰度 (`grayscale(100%)`)，Hover时恢复彩色，保持界面整洁。
